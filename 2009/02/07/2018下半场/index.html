2018下半场小结

Posted on 2009-02-07 | Edited on 2019-02-11 |
  • 又迟到了 hmmmmm。。其实题文无关（让我来改个时间把这篇文章藏起来

接着上半场来说吧，不过有好些事情都已经忘记了……

甚至也忘了是为什么忘记

这年夏天没有见面，梦里想象了千百次的场景连尝试都没有尝试。（尝试指订票）

岩石里的花 / 会开
离开的你 / 会回来？

偶尔刷了几次知乎，读男孩追女孩时女孩反感的事情，发现自己各种错误挨个犯了个遍…… 我总是要过好几年才会明白那时候的你…… 好难过

你对我 / 的好
我却错手 / 毁掉

Ingress 不怎么开了。感觉它是让人接触到有钱有闲的生活，然后清楚地意识到自己并没有能力实现这样的自由…… 太扎心了

我为什么，不好好读书……

另一方面，走在汴大的街头看手机不觉得亏大了吗……

至今很后悔和你出去玩的时候我在刷po……

在 lab 打杂，似乎添了不少乱却只做了一点点微小的工作，十分惭愧。真的是好棒的团队，明白了很多道理，能感觉得到自己秋天的状态和先前不太一样。当时 Jet 学长说我 ics 应该问题不大吧…… 没奶过来，不过比数分三连要好得多…… 借您吉言。感激！感谢学长学姐 carry 我，十分包容我这个菜狗 QAQ

开学之后花了大量的时间在某个 Project 上面，今年年初（过几周）就要 release 了，十分兴奋，希望可以无锅上线。

ics 是门好课，内容充实还调分。至于数算，院外的能不选别选了，这课实在是不知道在想什么，我甚至觉得卷子没有人校对过。

写完上半场总结后的一周，有天晚上下定决心要造 OI Wiki。（一直想要记下来可是好像又有点记不清细节了）。

谁又能想到一个破站如今已经六十万字了呢= =

感谢朋友们的支持和鼓励，不说了此处省略（都塞进那回造的 pdf 里了。

哦不过还是有点乱七八糟的槽可吐。比如那天晚上跳出来一万个人说“讲道理这个东西我来做更合适”，可是呢？我就不展开说有的团队几个月拿同一张截图忽悠人。这个wiki想要解决什么呢？又能够解决什么问题呢？motivation可能是见到一堆粗制滥造的网课，以及自己的学习经历过于煎熬，希望后来人可以少走自己的弯路？翻了下七月份的记录，当时可能想的是希望能产生的贡献是推动形成一个比较系统的知识库类似物，用来解决新入坑的玩家的问题：有没有什么系统学习 OI 的方法。说起来惭愧，后来遇到有人问，还是不好意思推荐 OI Wiki，觉得还不够精致。其实我觉得这种项目谁来做都无所谓，重要的是做出来真正有用的东西，对社区有确确实实的贡献，或大或小有就行，而不是有人想要做点什么贡献，就跳出来嘴炮，自己没能做成的事情也希望别人做不成。

再说林花，就好像是做了很久的一场梦吧，走通了所有流程，写好了整个框架，但是没有上线的想法了。可能是因为某种奇奇怪怪的原因刷了太久微博（17年时候还说现在哪有人用，真香），见到各种用户喷开发者的例子，觉得release一个服务出来，而又不能保证用户素质。纯粹是给自己添堵，还不如留着自用。另一方面，通过这半年的种种事情我意识到，OIer不过是一个 label，里面的形形色色林林总总也是参差不齐的，就比如汴大这所学校也会有盗刷学生卡的人跳出来…… 我觉得可能我并没有那么热爱 OI，更多的是在怀念那些年有梦可做的时光？可能也是想你了吧。

听说你读 data science 了。

我们是不是遇见得太早了。

我是不是摸鱼太久才会这么想念你？
