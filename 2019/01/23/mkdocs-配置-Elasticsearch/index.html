后端

用 express 封装了一下搜索和查询结果（main.js）

关于 cors，还是被坑了一下，在 oi-wiki.org 搜的时候，存到了 cache 里，再用别的域名搜的时候，就会被 cors 给拦住。需要加上 add_header 'Vary' "Origin";，来告诉 client side 这个 cors 的域名可能会有多种情况。
